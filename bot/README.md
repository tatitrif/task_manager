# Telegram Bot — Task Manager

Альтернативный интерфейс для управления задачами.

## Описание

Telegram-бот, который предоставляет альтернативный интерфейс для взаимодействия с задачами через API, обеспечивая удобство и доступность.

## Установка и запуск

```bash
# клонируйте репозиторий
git clone https://github.com/tatitrif/task_manager.git

# перейдите в директорию приложения
cd task_manager/bot

# настройте .env (пример в .env.example)
cp .env.example .env

# запустите приложение
uv run python main.py
```

## Структура проекта

- `bot/`
  - `main.py` - Точка входа, обработчики команд
  - `pyproject.toml` - Зависимости и метаданные
  - `README.md` - Документация
