# Telegram Bot — Task Manager

Альтернативный интерфейс для управления задачами через Telegram.

## Описание

Этот проект представляет собой Telegram-бот, который предоставляет альтернативный интерфейс для взаимодействия с задачами через API, обеспечивая удобство и доступность.

## Технологии

- **Telegram-бот**:
  - Библиотека: **Aiogram** для реализации бота.
  - Бот должен аутентифицировать пользователей и взаимодействовать с API приложения.

## Установка и запуск

```bash
# клонируйте репозиторий
git clone https://github.com/tatitrif/task_manager.git

# перейдите в директорию приложения
cd task_manager/bot

# настройте .env (пример в .env.example)
cp .env.example .env

# запустите приложение
uv run python main.py
```
