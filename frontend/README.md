# Task Management Web Application Frontend

## Описание

Фронтенд веб-приложения для управления задачами в команде.

## Функциональность

- Аутентификация пользователей (регистрация, вход, выход)
- Управление задачами
- Отображение задач в реальном времени через WebSocket
- Уведомления в реальном времени

## Структура проекта

- `frontend/`
  - `html/` - HTML шаблоны
    - `index.html` - отображение тасков
    - `login.html` - отображение формы логина
    - `register.html` - отображение формы регистрации
  - `js/` - JavaScript файлы
    - `auth.js` - управление аутентификацией
    - `config.js` - определяем конфигурацию API и WebSocket
    - `tasks.js` - работа с задачами
    - `ui.js` - обновление пользовательского интерфейса
    - `ws.js` - управление WebSocket соединениями

## Установка и запуск

1. Клонируйте репозиторий.
2. Перейдите в директорию приложения `cd frontend`
3. Запустите сервер `python -m http.server 5500`
4. Откройте `http://127.0.0.1:5500/html/index.html` в браузере
